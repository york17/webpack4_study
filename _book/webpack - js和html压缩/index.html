webpack - js和html压缩

webpack - js和html压缩

  • js压缩

只需要将webpack.config.js中的mode改成production生产环境即可自动压缩js。

  • html压缩

    使用的是之前的html-webpack-plugin

    增加一些配置项

    new HtmlWebpackPlugin({
  template: `./src/index.html`,
  minify: {
    // 移除空格
    collapseWhitespace: true,
    // 移除注释
    removeComments: true,
  }
}),

    添加minify指定规则即可。
